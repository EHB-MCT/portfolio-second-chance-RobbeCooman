Podcast Manager API Documentation

GET All Podcasts

Get a list of all podcasts.

GET /api/podcasts

Returns: An array of podcast objects.

    [
      {
        "id": 1,
        "title": "Sample Podcast",
        "description": "This is a sample podcast."
      },
      {
        "id": 2,
        "title": "Another Podcast",
        "description": "Another example podcast."
      }
      // ...
    ]

GET Single Podcast by ID

Get details of a single podcast.

GET /api/podcasts/:id

Returns: The podcast object matching the provided ID.

    {
      "id": 1,
      "title": "Sample Podcast",
      "description": "This is a sample podcast."
    }

POST New Podcast

Create a new podcast.

POST /api/podcasts

Body:

    {
      "title": "New Podcast",
      "description": "Description of the new podcast."
    }

Returns: The newly created podcast object.

    {
      "id": 3,
      "title": "New Podcast",
      "description": "Description of the new podcast."
    }

PUT (Update) Podcast by ID

Edit details of a podcast.

PUT /api/podcasts/:id

Body:

    {
      "title": "Updated Podcast",
      "description": "Updated description of the podcast."
    }

Returns: The updated podcast object.

    {
      "id": 1,
      "title": "Updated Podcast",
      "description": "Updated description of the podcast."
    }

DELETE Podcast by ID

Delete a single podcast.

DELETE /api/podcasts/:id

Returns: A message indicating successful deletion.

    {
      "message": "Podcast deleted successfully"
    }
